1. ### Подготвка окружения:
```
// Из корневой директории репозитория

// В любом случае:
python -m venv <имя_окружения>             // <- Создание виртуального окружения
.\<имя_окружения>\Scripts\activate         // <- Запуск виртуального окружения
python.exe -m pip install --upgrade pip    // <- Обновление менеджера пакетов (не обязательно)
pip install -r requirements.txt            // <- Установка зависимостей
Добавить папку с окружнием в .gitignore

// Если понадобится:
deactivate                                 // <- Выход из виртуального окружения

// У вас должен быть токен бота в телеграм

https://t.me/BotFatherhttps://t.me/BotFather // <- Получение токена

src/config/config.py dataclass conf        // <- Заменить пустую строку на свой токен 

```
2. ### Запуск:
```
// Из src
python bot.py                              // <- Запуск

// Если понадобится:
из терминала комбинацией ctrl+c            // <- Выключение

```
